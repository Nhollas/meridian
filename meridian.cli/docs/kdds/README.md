# Key Decision Discovery Documents

This directory records the important decision explorations behind the Meridian CLI.

KDDs are useful when the alternatives and trade-offs are likely to matter later, even if the final answer is already known today. They capture the path to the decision rather than just the settled outcome.

- [KDD-003: Deferring Process API Integration for the Proof of Concept](./KDD-003-deferring-process-api-integration-for-the-proof-of-concept.md)

## Relationship to Other Docs

- ADRs record the durable decisions that shape the current architecture.
- KDDs record the trade-offs and alternatives considered on the way to those decisions.
- The CLI reference remains the source of truth for current shipped behaviour.
